Repo with files for Portfolio Website 

Directory Structure
-------------------
* css\              - Stylesheets CSS files for the template.
* images\           - Images used in template.
* scripts\          - JavaScript used in the template.
* index.html        - Main HTML page to open the template in browser.
* portfolio.html    - Full portfolio page (backup).
* toggle-site.sh    - Script to switch between construction/portfolio modes.
* LICENSE           - License file.

Site Toggle
-----------
```bash
# Switch to portfolio
./toggle-site.sh portfolio

# Switch to construction page
./toggle-site.sh construction

# Check current status
./toggle-site.sh
```
